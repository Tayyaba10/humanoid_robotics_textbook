# Quickstart: Testing the Landing Page

1.  **Run the development server**:
    ```bash
    cd humanoid-robotics-textbook
    npm install
    npm start
    ```
2.  **Verify the landing page**:
    - Open your browser and navigate to `http://localhost:3000`.
    - You should see the new landing page with the title, description, and CTA buttons.

3.  **Verify the documentation**:
    - Navigate to `http://localhost:3000/docs`.
    - You should see the documentation modules.

4.  **Verify navigation**:
    - From the landing page, click the "Start Reading" button. You should be taken to `/docs/intro`.
    - From the landing page, click the "View Modules" button. You should be taken to `/docs`.
    - Use the navbar to navigate between the "Home" page and the "Docs" page.
